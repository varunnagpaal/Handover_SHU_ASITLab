\section{FTP 目录说明}
FTP 根目录位于服务器 C 盘的 FTP\_storage。

temp 目录：临时交互用。

pub 目录：主要目录，其中 asitlab 子文件内含各用户自己管理的文档。

incoming 目录：结构与 pub 类似。
